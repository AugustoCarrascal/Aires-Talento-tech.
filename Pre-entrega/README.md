# aires - E-commerce de Comida Rápida

## Propósito de la Página (Descripción)
Este proyecto es una landing page para un negocio digital de comidas rápidas llamado "Aires", desarrollada como entrega para el curso de Desarrollo Web. 

**El propósito principal de la página es:**
Ofrecer a los clientes un menú visual y directo de los productos principales, brindar información de contacto y ubicación del local, y recolectar pedidos mediante un formulario funcional. Todo en una interfaz rápida, limpia y adaptable a cualquier dispositivo móvil.

## Tecnologías Utilizadas
* **HTML5**: Estructura semántica (header, nav, main, section, footer).
* **CSS3**: Estilos personalizados y diseño responsivo.
* **Flexbox**: Utilizado para la alineación de las tarjetas de productos.
* **CSS Grid**: Implementado en la sección de reseñas de clientes.
* **Google Fonts**: Tipografía "Supermercado One".
* **Formspree**: Configuración funcional para el envío de datos del formulario.
